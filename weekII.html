Tydzień II

HTML

Hyper Text Markup Language - składa się ze znaczników/tagów/elementów; tworzy szkielet strony

Znaczniki:

  • main - główna część dokumentu
  • section - sekcja dokumentu
  • header - nagłówek dokumentu/sekcji
  • footer - stopka dokumentu/sekcji
  • h1-h6 - tekstowe nagłówki tytułowe dokumentu/sekcji
  • article - artykół / samodzielna treść
  • b , i - Nie mają dużego znaczenia semantycznego (nie są istotne dla teści) - służą do wyróżniania fragmentu z tekstu otaczającego (b-pogrubienie, i-kursywa)
  • - strong - pogrubienie, podkreśla wysoką istotność i poważność frazy w nim zawartej (istotne w czytnikach stron)
  • em - kursywa, kładzie mocniejszy nacisk na zawartość (istotne w czytnikach stron)
  • a - link/łącze wewnątrz dokumentu (a href="podstrona"> tekst /a
  • id - odnieśienie to fragmentu strony np. (section id="góra") ... (a href="#góra") Przejdź do góry /a
  • img - obrazek o podanym adresie (img scr="plik" alt="opis obrazka (czytnik)"

Atrybuty:

  • atrybuty są"niewidzialne" - chowamy je w znaczniku,
  • nadają lementom dodatkowe właściwości (herf, src)
  • pozwalają na grupowanie elementów (class)
  • pozwalają na identyfikowanie elementów (id)
  • można je miksować
  • spis atrybutów "globalnych" - strona w prezentacji

Validator - strona do sprawdzania poprawności kodu HTML

HTML - podsumowanie:

  • Opakowuje i opisuje treść strony
  • Mówi przeglądarce o strukturze dokumentu (i o tym co ma wczytać)
  • Nie jest językiem programowania, jest językiem znaczników (Markup Language)
  • Aktualną rekomendowaną wersją jest 5.2
  • można je miksować
  • Całą specyﬁkację znajdziesz na stronie w3c

CSS

Cascading Style Sheets - kaskadowe arkusze stylów

CSS wpływają na wygląd elementów HTML na ekranie,wydruku itp.

CSS - podsumowanie:

  • Odpowiadają za prezentację treści strony
  • Nie mają jednej wersji (rozwój modułów), potocznie mówi się, że aktualna wersja to 3
  • Przeglądarki potraﬁą różnie interpretować CSS co prowadzi do tego, że...
  • … pisanie stylów powoduje wiele frustracji u programistów (ale jest też niezłym wyzwaniem)
  • można je miksować
  • Całą specyﬁkację znajdziesz na stronie w3c
